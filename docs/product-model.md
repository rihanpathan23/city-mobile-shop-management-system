# Product Model

## Purpose

Stores all products available in the City Mobile Shop.

---

## Fields

- productId (Primary Key)
- productName
- category
- brand
- buyingPrice
- sellingPrice
- stockQuantity
- condition (New / Used)
- imeiNumber (Optional)
- description
- imageUrl
- createdAt
- updatedAt

---

## Categories

- Second Hand Phone
- Charger
- Cable
- Headphones
- Neckband
- Mobile Cover
- Screen Guard
- Car Charger
- Mobile Stand
- Adapter
- Other Accessories
